## 소개

이 프로젝트에서는 가상 시뮬레이션과 실제 응용 프로그램의 교차점을 경험하게 됩니다. 레벨 1 자율주행의 핵심 기능인 차선 유지 보조 시스템(LKA)의 메커니즘을 깊이 탐구하게 됩니다. 고급 시뮬레이션 플랫폼과 실제 하드웨어를 사용하여 가상과 실제 세계 모두에서 작동할 수 있는 LKAS를 설계하고, 테스트하며, 구현하게 됩니다. 
</br>


## 배경 정보

자율주행 자동차의 꿈은 기술적 현실이 되기 훨씬 전부터 수십 년 동안 존재해왔습니다. 1930년대 세계 박람회에서 선보인 초기 무선 조종 자동차부터 공상 과학 소설의 미래형 차량까지, 자율주행은 항상 인간의 상상력을 사로잡아 왔습니다. 차선 유지 보조 시스템은 이 꿈을 실현하기 위한 첫 번째 단계 중 하나입니다. 이것은 단순한 기술적 경이로움이 아니라, 차선 이탈 사고를 줄임으로써 더 안전한 도로를 보장하는 데 중요한 역할을 합니다.  
</br>


## 목표

1. 시뮬레이션된 센서를 사용하여 차선 표시를 감지하고 추적할 수 있는 가상 LKAS를 설계합니다.
2. 의도하지 않은 차선 이탈 시 가상 운전자에게 경고하는 피드백 메커니즘을 구현합니다.
3. 가상 차량이 차선을 유지할 수 있도록 조향 또는 제동 조정과 같은 교정 개입을 시뮬레이션합니다.
4. PiRacer를 사용하여 가상 LKAS 시스템을 실제 응용 프로그램으로 변환하고, 실제 센서와 환경에서 효과적으로 작동하도록 합니다.

</br>


## 필수 과제

1. CARLA 또는 Gazebo와 같은 플랫폼을 사용하여 포괄적인 시뮬레이션 환경을 구축합니다. 이 설정에는 상세한 테스트 트랙과 필요한 모든 시뮬레이션 센서가 장착된 차량이 포함되어야 합니다.
2. 차선 표시를 식별하고 이에 대한 차량의 위치를 결정하는 알고리즘을 개발합니다.
3. 차량이 차선을 벗어나면 시각 및 청각 신호를 모두 사용하여 운전자에게 경고하는 피드백 시스템을 구현합니다.
4. 차량을 차선으로 되돌리기 위한 시뮬레이션 개입을 생성합니다.
5. PiRacer에 LKAS를 설정하여 시뮬레이션에서 실제 응용 프로그램으로 전환합니다. 필요한 센서를 통합하고 실제 환경에서 작동하도록 보정합니다.

</br>
